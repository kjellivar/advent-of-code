## Running:

```shell
$ npm install
$ npm start
```

To create files for a new day, run:

```shell
$ npm run new --date=<year>-<day>
```

Example for creating a new js file and input txt for 2022 day 4:

```shell
$ npm run new --date=2022-04
```
